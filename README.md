# Projeto de Interface – Figma

Este projeto foi desenvolvido durante o curso **Conhecendo o Figma**, com o objetivo de introduzir e praticar os principais conceitos de **design de interface (UI)** utilizando a ferramenta Figma.
O layout criado simula um **feed de publicações**, comum em aplicações e plataformas digitais, permitindo explorar a organização visual e a experiência do usuário.

## Sobre o curso

O curso **Conhecendo o Figma** apresenta os fundamentos da ferramenta, abordando desde a criação de frames até a organização de componentes e estilos visuais.
Durante o curso, este projeto foi utilizado como prática para aplicar os conceitos aprendidos.

## Objetivo do projeto

- Compreender o funcionamento do Figma
- Criar interfaces visuais organizadas
- Trabalhar com hierarquia de informações
- Explorar layouts de feed
- Desenvolver noções iniciais de UI/UX
- Praticar alinhamento, espaçamento e consistência visual

## Principais elementos da interface

- Barra lateral de navegação com ícones
- Campo de busca no topo
- Cards de publicações com imagem, título e descrição
- Ícones de interação (curtidas, comentários, compartilhamentos)
- Layout em modo escuro (dark mode)
